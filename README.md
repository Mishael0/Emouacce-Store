 Emouacce Store

A modern e-commerce platform built with React, Redux, and Tailwind CSS.

Features
- Product browsing and search
- Shopping cart & wishlist
- Order management system
- Responsive design
- Real-time validation

 Live Demo
[Visit Emouacce Store](https://emouacce-store.vercel.app)

 Tech Stack
- React 18
- Redux Toolkit
- React Router
- Tailwind CSS
- Lucide Icons

 Installation
```bash
npm install
npm run dev
```
